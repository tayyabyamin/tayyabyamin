Spelling Test Application
=======================

System Requirements:
- Windows 10 or later
- Microsoft Word (for .docx file support)
- Windows Text-to-Speech components (usually pre-installed)

Installation Instructions:
1. Copy the entire "installation file spelling test" folder to your desired location
2. Double-click "Spelling Test.exe" to run the application
3. If prompted by Windows SmartScreen, click "More info" and then "Run anyway"

Features:
- Text-to-speech word pronunciation
- Spelling verification
- Adjustable speech speed and volume
- Multiple voice options
- Progress tracking

How to Use:
1. The application will read words from "words.docx"
2. Enter the spelling you hear in the text box
3. Click "Submit" to check your spelling
4. Use "Repeat Word" to hear the word again
5. Click "Next" for a new word
6. Adjust speed and volume using the sliders
7. Select different voices from the dropdown menu

Troubleshooting:
1. If the application doesn't start:
   - Make sure you have Microsoft Word installed
   - Check if Windows Text-to-Speech is enabled in Windows Settings
   - Try running the application as administrator

2. If you can't hear the words:
   - Check your system volume
   - Make sure your speakers/headphones are properly connected
   - Try selecting a different voice from the dropdown menu

3. If you get an error about missing files:
   - The application will automatically create default files
   - You can add your own words to words.docx using Microsoft Word

Your progress is automatically saved in "correct_spellings.txt"

For any issues or questions, please contact the developer.
